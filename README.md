# AI Agent vs. Human: Cost of Work Calculator

A self-contained, interactive calculator comparing the fully-loaded annual cost of a human hire
(salary, benefits, 401(k) match, payroll taxes, PTO/equipment/recruiting) against the running cost
of an AI agent (tokens, software, compute, skill development). Built by Cirrius Solutions.

Single `index.html`, no dependencies. Framing: augmentation, not replacement.
